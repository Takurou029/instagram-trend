# Instagram Trend 分析ツール 仕様書

## プロジェクト概要
Instagramのハッシュタグや特定のアカウントのエンゲージメントを分析し、トレンドの把握や競合分析を効率化するためのWebアプリケーションです。

## 技術スタック
- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS / Inline Styles
- **Deployment**: Vercel (GitHub連携による自動デプロイ)
- **API**: Instagram Graph API (Business Discovery / Hashtag Search)

## 主要機能

### 1. アカウント比較分析
最大2つのアカウントを並べて運用パターンを比較分析します。
- **履歴・URL解析機能（NEW）**: 
  - InstagramのプロフィールURL（例: `https://www.instagram.com/username/`）を直接貼り付け可能。
  - 過去に比較したアカウントは「履歴リスト」として保存され、次回からワンクリックで呼び出し可能。
- **日次推移グラフ**: 直近30日間の「いいね数」または「投稿本数」の推移を折れ線グラフで表示。
- **月次合計グラフ**: 今月と先月の「いいね数」または「投稿本数」の合計を棒グラフで比較。
- **指標切り替え**: グラフ右上のスイッチで「いいね数」と「投稿本数」を切り替え。※「再生数」はAPI制限により不正確なため、より実用的な「投稿本数」に刷新。
- **最多いいね Top 3 投稿**: 直近100件の中から最も反響のあった投稿を表示。反応率（いいね/フォロワー数）も算出。

### 2. トレンド調査（ハッシュタグ分析）
指定したハッシュタグで「今、勢いがある」投稿を独自のアルゴリズムで抽出します。
- **精度重視のフィルタリング（NEW）**: いいね数が非公開（APIで0と返る）の投稿を自動で除外。正確な数値に基づいた分析のみを提供。
- **急上昇スコア (Velocity)**: 「(いいね数 + コメント数×2) ÷ 投稿からの経過時間」で算出。単なる「いいね順」ではなく、短時間で伸びている投稿を上位表示。
- **リール動画プレビュー（NEW）**: 静止画サムネイルに代わり、動画そのものを表示。マウスホバーで動画が自動プレビュー再生され、内容の把握が容易。
- **CSVエクスポート**: 抽出した投稿リストをCSV形式でダウンロード可能。

## API制限と注意事項
- **いいね数の非公開設定**: 投稿者がいいね数を非公開にしている場合、APIからは「0」として返されるため、本ツールでは精度の観点からリストに表示されません。
- **ハッシュタグ検索の範囲**: 1回のリサーチで最大50件（Top 25件 + Recent 25件）をマージして分析します。
- **レート制限**: Instagram API側の制限により、短時間の過度な連続検索は一時的に制限される場合があります。

## 運用・更新方法
- **自動デプロイ**: GitHubの `main` ブランチにコードをプッシュすると、Vercelへ自動的に反映されます。
- **環境変数**:
  - `INSTAGRAM_ACCESS_TOKEN`
  - `INSTAGRAM_APP_SECRET`
  - `INSTAGRAM_BUSINESS_ACCOUNT_ID`
